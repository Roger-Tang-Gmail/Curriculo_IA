# Instalação de IA com Prompt

## Instalando o Gemini

``` Console
> npm install -g @google/gemini-cli
```
## Rodando Gemini
``` Console
>  gemini
```

## Pré-Requisitos. 
Necessário a versão 20 do node.
## Se der erro
> nvm list available

### Instala a versão 20 no node 
> nvm install 20
> nvm use 20.20.2
> node --version
