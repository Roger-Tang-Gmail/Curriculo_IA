# Gemini CLI - Workspace Instructions

Este arquivo contém as instruções e mandatos fundamentais para o funcionamento do Gemini CLI neste projeto (`IA`).

## 🎯 Contexto do Projeto
Este workspace é dedicado a experimentos e configurações relacionadas a Inteligência Artificial, especificamente utilizando o `gemini-cli`.

## 🛠️ Ambiente de Desenvolvimento
- **Node.js**: Versão 20 (recomendada: 20.20.2).
- **Gerenciador de Pacotes**: npm.

## 📜 Mandatos do Workspace
1. **Padrões de Documentação**: Mantenha os arquivos `.md` organizados e em português (PT-BR), conforme os arquivos existentes (`Instalação.md`).
2. **Segurança**: Nunca inclua chaves de API ou segredos diretamente nos arquivos do repositório. Use variáveis de ambiente se necessário.
3. **Instalação**: Sempre verifique a versão do Node.js antes de executar comandos globais do Gemini, conforme instruído em `Instalação.md`.

---
*Nota: Este arquivo é lido pelo Gemini CLI em cada sessão para garantir que as instruções específicas do projeto sejam seguidas.*
